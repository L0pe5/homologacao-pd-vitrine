Para que a PD Vitrine funcione corretamente, é necessário que os usuários adicionem algumas informações no GitLab.
 
1) Crie um repositório que o nome seja o seu username (exemplo: leonardo.maciel, emanuel.bravo...) e adicione um README pessoal (descrição para a seção "Quem sou eu?" da página de perfil).
 
2) Envie para o GitLab os projetos que deseja exibir na página de perfil da PD Vitrine (no máximo 4 projetos).
 
3) Adicione uma pasta chamada screenshots na raiz do projeto, contendo as imagens em formato .png, que serão usadas na página do projeto para ilustrá-lo.
 
4) Adicione descrição nos projetos (Project Settings -> Project Description)
 
5) Adicione uma bio no perfil do GitLab (a descrição do card de perfil é puxada de lá). Também adicionar o link do LinkedIn no perfil do GitLab.
 
6) Para adicionar o link de "aplicação" na página do projeto, acesse o projeto no GitLab e vá em Settings → General → Badges.
Crie uma Badge com:
Name: deploy
Link: o endereço da aplicação rodando
URL da Badge: pode ser qualquer uma. Caso deseje, pode usar esta da Vercel (obrigatória a presença da Badge):
https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white
 
1) Crie um novo repositório com o nome no formato nome.sobrenome.formacoes.
Exemplo: leonardo.maciel.formacoes
Esse repositório deve conter um README com o conteúdo da seção "Formações" da página de perfil.
